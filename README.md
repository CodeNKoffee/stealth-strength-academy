# Stealth Strength Academy

> **Become the Person You've Always Desired.**

Stealth Strength Academy is a premium coaching platform designed to transform your mindset, habits, and physical strength. Led by Hatem Soliman, the academy focuses on the philosophy that physical strength and mental resilience are two sides of the same coin.

## 🚀 Overview

This repository contains the landing page for Stealth Strength Academy, featuring:
- **3-Month Transformation Program**: A comprehensive coaching journey from foundation to mastery.
- **7-Day Bodyweight Strength Guide**: A free starter guide to build momentum with no equipment.
- **1-on-1 Coaching**: Personalized video sessions and direct accountability.

## ✨ Key Features

- **Personalized Coaching**: Weekly 1-on-1 sessions tailored to individual goals.
- **Holistic Approach**: Focuses on body, mindset, work performance, and relationships.
- **No-Gym Starter Guide**: Progressive bodyweight workouts that take less than 12 minutes daily.
- **Accountability Systems**: Mid-week check-ins and direct messaging access.
- **Whop Integration**: Seamless booking and checkout via Whop.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Custom design system with a premium "dark mode" aesthetic, using CSS variables and modern layout techniques (Grid/Flexbox).
- **JavaScript**: Minimal, efficient logic for interactive elements like FAQs.
- **Google Fonts**: Utilizing *Bebas Neue* for headings and *DM Sans* for body text.

## 🏁 Getting Started

To view the landing page locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/CodeNKoffee/stealth-strength-academy.git
   ```
2. Open `index.html` in any modern web browser.
